## Installation
1. Clone the repository:
```bash
git clone https://github.com/AyushKashyapII/Pacman.git
cd Pacman
```

2. Install the required packages:
```bash
pip install -r requirements.txt
```

## Running the Game
To start the game, run:
```bash
python pacman.py
```

## Development
